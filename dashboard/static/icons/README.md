# Science Pack Icons

This directory should contain the science pack icon files. 

## How to Obtain Icons

You can obtain official Factorio science pack icons from these sources:

1. **From the Factorio game files** (if you own the game):
   - Navigate to your Factorio installation directory
   - Look in `data/base/graphics/icons/`
   - Copy the science pack PNG files

2. **From the Factorio Wiki**:
   - Visit: https://wiki.factorio.com/Category:Game_images
   - Download individual science pack images

3. **From the icons-factorio GitHub repository**:
   - Visit: https://github.com/deniszholob/icons-factorio
   - Use their icon viewer or download the repository

## Required Icon Files

The following icon files should be placed in this directory:

- `automation-science-pack.png`
- `logistic-science-pack.png`
- `military-science-pack.png`
- `chemical-science-pack.png`
- `production-science-pack.png`
- `utility-science-pack.png`
- `space-science-pack.png`
- `metallurgic-science-pack.png`
- `electromagnetic-science-pack.png`
- `agricultural-science-pack.png`
- `cryogenic-science-pack.png`
- `promethium-science-pack.png`

## Icon Format

- Format: PNG with transparency
- Recommended size: 64x64 pixels
- Alternative: SVG format is also supported

## Fallback

If icons are not available, the dashboard will display colored circles based on the science pack type.
